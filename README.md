# Development
```
$ node -v
v12.0.0

$ npm install

$ npm start
```

# Make app icons
```
$ npm i -g electron-icon-maker

$ ./make-icon.sh
```

# Packaging
```
$ npm run package-osx

$ npm run package-linux

$ npm run package-win
```
