# Twitter Mini

Multi-platform Twitter Client built with Electron

# Development
```
$ node -v
v12.0.0

$ npm install

$ npm start
```

# Packaging
```
$ npm run package-mac
```

# app icons
```
$ npm i -g electron-icon-maker

$ electron-icon-maker --input=./assets/twitter-mini.png --output=./assets/icons
```
